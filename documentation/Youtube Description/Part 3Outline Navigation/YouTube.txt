vsCode Pt 3 Outline and brace Navigation

How to navigate large amounts of code using Outline view and matching braces navigation, including parentheses, brackets, braces, and angle brackets. Using the keys ctrl + shift + o and ctrl + shift + backslash.
We as blind coders process coding one line at a time and fail to see those helpful hints decorating the screens that our sighted counterparts see.
This is part of a series of videos on using vsCode with a screen Reader, such as Jaws and NVDA.  This series will outline  the tips and tricks I as a blind coder have figured out for myself. This will supplement other resources for all coders (blind or sighted).
